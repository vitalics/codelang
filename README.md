# codelang

A compiled programming language inspired by Python, Rust, and Jai.

## Status: Phase 1 — `print("Hello, World!")`

The compiler pipeline is working end-to-end:

```
.code source  →  Langium parser  →  LLVM IR text  →  clang  →  native binary
```

## Quick start

```sh
npm install
npm run dev          # langium:generate + tsc

# Compile and run
node bin/codelang.js run examples/hello.code
# or the .code files at the repo root:
node bin/codelang.js run simple.code
```

## Syntax (Phase 1)

```codelang
# This is a comment

const fn main() {
  print("Hello, World!");
}
```

- **`const fn name()`** — function declaration
- **`print(expr)`** — print a string followed by a newline
- Comments: `#`, `//`, `/* */`

## Compiler commands

| Command                                    | Description                                              |
| ------------------------------------------ | -------------------------------------------------------- |
| `node bin/codelang.js compile <file>`      | Compile to native binary (produces `<file>.ll` + binary) |
| `node bin/codelang.js compile <file> --ir` | Emit LLVM IR only, skip clang                            |
| `node bin/codelang.js run <file>`          | Compile and immediately run                              |
| `-d <dir>` option                          | Set output directory for artefacts                       |

## Project layout

```
src/
  language/
    codelang.langium      ← grammar (Langium DSL)
    codelang-module.ts    ← Langium service wiring
    generated/            ← auto-generated by `langium generate` (do not edit)
  cli/
    main.ts               ← CLI entry (Commander.js)
    cli-util.ts           ← document extraction helper
    ir-generator/
      index.ts            ← LLVM IR text emitter
bin/
  codelang.js             ← CLI binary wrapper
examples/
  hello.code              ← canonical hello-world example
```

# Features

- [~] Safe for memory — `Option<T>` / `Result<T,E>` force explicit error handling; `using` / `defer` for deterministic cleanup; use-after-free detection in C runtime; safe collection accessors (`getSafe`, `atSafe`, `popSafe`, …). Missing: compile-time borrow checker, full GC.
- [] Safe for concurrency — planned (actors / channels)
- [] Safe for garbage collection — planned (arena or tracing GC); currently `using`/`free()` manual
- [x] `using` keyword
- [x] Type inference
